# Shabu Je Chan - ระบบสั่งอาหารร้านอาหาร

ระบบสั่งอาหารร้านอาหารแบบเว็บที่ทันสมัย สร้างด้วย React และ Vite ระบบนี้ให้โซลูชันครบวงจรสำหรับการจัดการร้านอาหาร รวมถึงการสั่งอาหารของลูกค้า การจัดการพนักงาน และแดชบอร์ดสำหรับผู้ดูแลระบบ

## 📋 ภาพรวมของระบบ

นี่คือแพลตฟอร์มระบบสั่งอาหารในร้านอาหารที่ครบครัน รองรับผู้ใช้หลายบทบาท:

- **ลูกค้า (Customers)** - เรียกดูเมนู สั่งอาหาร และจัดการตะกร้าสินค้า
- **พนักงาน (Employees)** - จัดการคำสั่งซื้อและบริการลูกค้า
- **ผู้ดูแลระบบ (Administrators)** - จัดการ_operations ของร้านอาหารและดูแดชบอร์ด

## 🛠️ เทคโนโลยีที่ใช้

### Frontend
- **React 19** - ไลบรารี UI
- **Vite** - เครื่องมือ build และ dev server
- **React Router** - การนำทางฝั่งไคลเอนต์
- **Lucide React** - ไลบรารีไอคอน
- **React Toastify** - ระบบการแจ้งเตือน
- **QR Code React** - การสร้างรหัส QR
- **ESLint** - การตรวจสอบโค้ด

## 🚀 คุณสมบัติ

### คุณสมบัติสำหรับลูกค้า
- เรียกดูเมนูร้านอาหาร
- เพิ่มสินค้าลงในตะกร้า
- สั่งอาหารตามหมายเลขโต๊ะ (`/order/:tableNumber`)
- จัดการตะกร้าสินค้า (`/cart/:tableNumber`)
- นัดหมายเวลาสั่งอาหาร (`/time/:tableNumber`)

### คุณสมบัติสำหรับพนักงาน
- แดชบอร์ดพนักงาน
- จัดการคำสั่งซื้อ
- เครื่องมือบริการลูกค้า

### คุณสมบัติสำหรับผู้ดูแลระบบ
- แดชบอร์ดผู้ดูแลระบบ
- ภาพรวมระบบและวิเคราะห์ข้อมูล
- จัดการร้านอาหารแบบครบวงจร

### การตรวจสอบสิทธิ์
- ระบบเข้าสู่ระบบสำหรับผู้ใช้ทุกบทบาท

## 📦 การติดตั้งและตั้งค่า

### ข้อกำหนดเบื้องต้น
- Node.js (v18 ขึ้นไป)
- npm หรือ yarn

### ติดตั้ง Dependencies
```bash
cd frontend
npm install
```

### เซิร์ฟเวอร์สำหรับพัฒนา
```bash
npm run dev
```

### สร้างสำหรับ Production
```bash
npm run build
```

### ดูตัวอย่าง Production Build
```bash
npm run preview
```

### ตรวจสอบโค้ด
```bash
npm run lint
```

## 🌐 เส้นทาง (Routes)

| เส้นทาง | Layout | คำอธิบาย |
|--------|--------|----------|
| `/` | WebLayout | หน้าแรกสาธารณะ |
| `/login` | - | หน้าเข้าสู่ระบบ |
| `/order/:tableNumber` | CustomerLayout | เมนูและสั่งอาหารตามโต๊ะ |
| `/cart/:tableNumber` | CustomerLayout | ตะกร้าสินค้า |
| `/time/:tableNumber` | CustomerLayout | นัดหมายเวลาสั่งอาหาร |
| `/admin` | AdminLayout | แดชบอร์ดผู้ดูแลระบบ |
| `/employes/home` | EmployeLayout | แดชบอร์ดพนักงาน |

## 🎨 โครงสร้างคอมโพเนนต์

### เลย์เอาต์
- **WebLayout** - เลย์เอาต์เว็บไซต์สาธารณะ
- **CustomerLayout** - เลย์เอาต์หน้าสำหรับลูกค้า
- **AdminLayout** - เลย์เอาต์แผงควบคุมผู้ดูแล
- **EmployeLayout** - เลย์เอาต์พอร์ทัลพนักงาน

### การจัดการสถานะ
- **CartContext** - จัดการสถานะตะกร้าสินค้าส่วนกลางด้วย React Context API
